# Event Photos

Add your conference and event photos here with the following naming convention:

- `devcon-sea-1.jpg`, `devcon-sea-2.jpg` - Ethereum Devcon SEA photos
- `devfest-uyo-1.jpg` - DevFest Uyo photos
- `eth-frontiers-1.jpg` - Ethereum Frontiers photos
- `wikimedia-1.jpg` - Wikimedia Hackathon photos
- `devfest-owerri-1.jpg` - DevFest Owerri photos
- `pycon-africa-1.jpg` - Pycon Africa photos

You can add more photos by updating the `speakingEngagements` array in `src/lib/data.ts`.

Recommended image size: 1200x800 or similar landscape ratio.
